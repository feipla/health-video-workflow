# 泛健康科普视频自动化工作流 🌿

> 输入一个 YouTube 健康视频链接，全自动产出 5-7 分钟中文科普视频（含白板手绘动画）

一键体验：**粘贴链接 → 自动出视频**

## 快速部署（到魔搭社区，国内秒开）

### 方式一：从 TRAE 一键部署（推荐）

## 配置 API Key（可选但强烈推荐）

脚本生成质量取决于 DeepSeek API：

1. 注册 https://platform.deepseek.com （免费送 500 万 tokens）
2. 在魔搭创空间"设置 → 环境变量"中添加：
   - 变量名: `LLM_API_KEY`
   - 变量值: `sk-xxxxxxxxxxxx`
3. 重启创空间生效

不配置也能用，会输出通用模板脚本。

## 工作流（7步全自动）

```
① YouTube链接 → ② 扒取信息 → ③ 转写英文 → ④ 生成中文脚本
    → ⑤ AI配音 → ⑥ 自动生成信息图 → ⑦ 白板手绘动画 → 最终视频
```

## 技术栈

| 步骤 | 工具 | 用途 | 费用 |
|------|------|------|------|
| 获取信息 | yt-dlp | 下载 YouTube 元数据 + 音频 | 免费 |
| 语音转写 | Whisper base | 英文语音 → 文本 | 免费 |
| 脚本生成 | DeepSeek API | 中文科普脚本 | 免费（500万 tokens） |
| AI 配音 | Edge TTS | 微软自然中文语音 | 免费 |
| 信息图生成 | Pillow | 脚本内容→多张信息图 | 免费 |
| 白板动画 | OpenCV + 手绘引擎 | 信息图→手绘动画视频 | 免费开源 |
| 视频合成 | FFmpeg | 白板片段+配音+字幕 | 免费 |
| Web 界面 | Gradio | 浏览器交互 | 免费 |

## 注意事项

- ⚠️ 内容仅限**健康生活方式科普**，不涉及疾病诊断/治疗
- ⏱ 每个视频处理约 10-20 分钟（Whisper 转写 + 白板动画最耗时）
- 💾 输出文件保存在临时目录，完成后及时下载
- 🔑 推荐配置 LLM_API_KEY 环境变量提升脚本质量
